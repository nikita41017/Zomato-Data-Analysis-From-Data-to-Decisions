# Zomato-Data-Analysis-From-Data-to-Decisions

📌 Project Summary 

This project analyzes Zomato restaurant data to uncover the key factors that influence restaurant ratings, customer preferences, and pricing strategies. The focus is on generating actionable insights that can support business decisions in the food industry.

🎯 Problem Statement

In a highly competitive food industry, restaurants need to understand:

What drives higher ratings?
Which cuisines attract more customers?
How pricing affects customer perception

This project answers these questions using data-driven analysis.

🛠️ Tools & Technologies
Python
Pandas & NumPy for data manipulation
Matplotlib & Seaborn for visualization

⚙️ Data Preparation

Cleaned missing and inconsistent data
Standardized categorical values
Converted data types for analysis
Removed duplicate entries

📊 Analysis Performed

Rating distribution analysis
Cost vs Rating relationship
Cuisine popularity trends
Location-based restaurant analysis
Customer engagement (votes) analysis

📈 Key Insights

Pricing alone does not determine ratings — customer experience plays a major role
Certain cuisines dominate market demand
Mid-range priced restaurants receive higher engagement
Location significantly impacts restaurant visibility and success

🚀 Business Impact

The insights from this analysis can help:

Restaurant owners optimize pricing strategies
Identify high-demand cuisines
Improve customer targeting based on location trends
